# Healing Stats Localization Agent Brief

This project localizes the Guild Wars 2 arcdps Healing Stats addon UI.

## Source Priority

1. `localization/canonical/Healing_Stats.en.ini` is the translation source.
2. `Healing_Stats.ko.ini` is a reviewed semantic reference for ambiguous terms only.
3. External GW2/MMORPG terminology research may be used for unclear meanings.

Do not translate from Korean directly. Translate from English, using Korean only as a clue for meaning.

## Product Context

This is an overlay/addon for Guild Wars 2 combat healing statistics. It shows healing, barrier generation, event ticks, combat time, sorting options, window display options, and optional live sharing of healing stats between squad members who use the addon.

## Fixed Policy

- Do not translate skill names.
- Do not translate profession/class names.
- Do not translate elite specialization names.
- Keep technical tokens such as `EVTC`, `RPC`, `TLS`, `kiB/s`, `Guild Wars 2`, `arcdps`, and file paths readable.
- Preserve every placeholder exactly: `{1}`, `{2}`, `%s`, `%u`, `%llu`, `\n`, `###HEALING_STATS_UPDATE`.
- Keep compact UI labels short. Do not turn small menu labels into full sentences.
- Translate tooltips naturally, but preserve their technical meaning.

## High-Risk Terms

- `Peers outgoing`: data from other squad members using live stats sharing. It is not "shared healing" as a game mechanic.
- For compact `Peers outgoing` labels, prefer other-squad-member / squad-member / peer-stat wording. Avoid "shared healing" wording or short labels built around "share/shared" if that could read as a GW2 healing mechanic.
- `Outgoing healing`: healing done/output by the measured player.
- `Barrier generation`: Guild Wars 2 barrier generated by the player. It is not generic shield/block.
- `Hits`: healing/barrier event ticks in this addon context. It is not necessarily weapon hit count.
- `Healing per hit`: healing per healing event/tick.
- `Barrier generation per hit`: barrier generated per barrier/healing event/tick.
- `Casts`: cast count, currently not implemented.
- `From Damage Dealt`: addon aggregate/pseudo entry for indirect healing derived from damage dealt. It is not a skill name.
- `Group`: party subgroup in GW2.
- `Squad`: GW2 squad.
- `Off-squad`: entities outside the squad.
- `Live stats sharing`: addon feature for sharing stats through a server with other addon users.

## Output Required From Each Language Agent

For each assigned language:

1. Create a complete `Healing_Stats.<lang>.ini` with exactly the same sections and keys as the English canonical file.
2. Preserve all placeholders and escaped newlines.
3. Provide a short review note listing any uncertain terms and the reasoning used.
4. Mark status as one of:
   - `machine_draft`
   - `reviewed_draft`
   - `needs_native_review`

These files are drafts until structural checks and final review pass.
