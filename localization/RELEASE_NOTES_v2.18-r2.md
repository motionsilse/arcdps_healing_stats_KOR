# Healing Stats Multilingual Pack v2.18-r2

## English

Unofficial multilingual localization build for Guild Wars 2 arcdps Healing Stats.

This release uses the original addon's English UI as the fallback and adds
external language INI files. The default setting is Korean, but the language can
be changed in `Healing_Stats.locale.ini`.

### Download

Download this ZIP file from the release:

```text
Healing_Stats_multilingual_v2.18-r2.zip
```

### Install

Put the files in:

```text
Guild Wars 2\addons\arcdps\
```

### Language Setting

Default:

```ini
[localization]
language=ko
```

To change the language, edit:

```text
Guild Wars 2\addons\arcdps\Healing_Stats.locale.ini
```

Supported language codes:

```text
ko  Korean
en  English, built-in original UI
de  German
es  Spanish
fil Filipino
fr  French
id  Indonesian
ja  Japanese
ms  Malay
pl  Polish
pt  Portuguese
ru  Russian
th  Thai
tr  Turkish
vi  Vietnamese
```

Use `language=en` to use the original built-in English UI. No English INI file
is needed.

### Notes

- Skill, profession, and elite specialization names are kept in English.
- Translations are conservative and stay close to the original English meaning.
- `Peers outgoing` means outgoing stats from other squad members through live stats sharing. It is not a shared-healing combat mechanic.
- This is an unofficial localization build and not an official build from the original author.

### Verification

- All non-Korean multilingual INI files passed strict key, placeholder, and escaped-newline verification.
- Korean plus all multilingual INI files passed key and placeholder verification.

## 한국어

Guild Wars 2 arcdps Healing Stats용 비공식 다국어 현지화 빌드입니다.

이 릴리즈는 원본 애드온의 영어 UI를 기본 fallback으로 사용하고, 외부 언어 INI

바꿀 수 있습니다.

지원 언어 코드:


`language=en`으로 설정하면 애드온에 내장된 영어 원문 UI를 사용합니다. 영어 INI
파일은 필요하지 않습니다.

### 참고

- 스킬명, 직업명, 엘리트 전문화명은 영어로 유지합니다.
- 번역은 영어 원문의 의미를 최대한 해치지 않는 방향으로 유지했습니다.
- `Peers outgoing`은 실시간 통계 공유를 통해 받은 다른 스쿼드원의 outgoing 통계입니다. 공유 치유 전투 메커니즘이 아닙니다.
- 원 제작자의 공식 빌드가 아닌 비공식 현지화 빌드입니다.

### 검증

- 한국어를 제외한 모든 다국어 INI 파일은 key, placeholder, escaped newline strict 검증을 통과했습니다.
- 한국어와 모든 다국어 INI 파일은 key와 placeholder 검증을 통과했습니다.
